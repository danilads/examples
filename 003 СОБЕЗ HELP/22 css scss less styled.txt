Less - это CSS-препроцессор.
Как работает: пишешь код с расширениями (переменные, вложенность, миксины и т.д.), потом он компилируется в обычный CSS.

Sass / SCSS - это тоже CSS-препроцессор, старше и мощнее чем Less.
Как работает: также компилируется в CSS.

Styled Components - это библиотека для React
Как работает: CSS пишется прямо внутри JavaScript
