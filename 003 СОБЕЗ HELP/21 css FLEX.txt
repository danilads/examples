flex-direction – направление главной оси
    row (по умолчанию) → элементы в строку слева направо
    row-reverse → справа налево
    column → сверху вниз
    column-reverse → снизу вверх

justify-content – выравнивание по главной оси
    flex-start (по умолчанию) – в начале
    flex-end – в конце
    center – по центру
    space-between – равномерно, с пустотой между
    space-around – с отступами вокруг
    space-evenly – равные отступы везде

align-items – выравнивание по поперечной оси
    stretch (по умолчанию) – растягивает по высоте контейнера
    flex-start – прижимает к началу
    flex-end – к концу
    center – по центру
    baseline – по линии текста

flex-wrap – перенос элементов
    nowrap (по умолчанию) – всё в одну строку
    wrap – перенос на новую строку
    wrap-reverse – перенос в обратном направлении

align-content – управление расстоянием между строками (работает только если есть перенос)

////

flex-grow – насколько элемент может расти, чтобы заполнить свободное место
    0 (по умолчанию) – не растёт
    1 – растёт пропорционально

flex-shrink – насколько элемент может сжиматься, если места не хватает
    1 (по умолчанию) – сжимается
    0 – не сжимается

flex-basis – "базовый размер" элемента до распределения свободного пространства
    может быть auto или (по умолчанию) 
    конкретное значение (200px, 50%)


flex: 1; /* flex-grow: 1; flex-shrink: 1; flex-basis: 0; */
