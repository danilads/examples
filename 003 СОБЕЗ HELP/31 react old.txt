1) constructor() → инициализация state, биндинг методов.

2) render() → возвращает JSX.

3) componentDidMount() → вызывается один раз после того, как DOM уже вставлен.
  - запросы к API,
  - подписки (события, WebSocket),


-) componentDidUpdate(prevProps, prevState) → вызывается при ререндере (но не первый раз)
  - реагировать на изменения конкретных props/state,
  - запускать побочные эффекты,
  - синхронизировать данные.

  componentWillUnmount() {
    console.log("Компонент удалён");
  }


-)  componentWillUnmount() → вызывается один раз перед удалением.
  - очистить таймеры,
  - отписаться от событий/сокетов,
