JavaScript — однопоточный язык (он выполняет код последовательно, строка за строкой).
Но у нас есть асинхронные операции (Микрозадачи, Макрозадачи)


1) Есть Call Stack (стек вызовов) — туда попадает весь синхронный код.

Event Loop бесконечно проверяет:
Если стек пустой (Call Stack) → забирает задачи асинхронные


2) Микрозадачи
Promise.then / catch / finally

3) Макрозадачи (macrotasks)
setTimeout
setInterval
